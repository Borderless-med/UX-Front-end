HOW TO EXPORT PDF:

  1. Press Ctrl+P to open Print dialog
  2. Select "Save as PDF" or "Microsoft Print to PDF"
  3. Click "More settings"
  4. Set Paper size: Custom (96 x 60 mm)
  5. Set Margins: None
  6. Set Scale: 100% (Default)
  7. Enable "Background graphics"
  8. Click "Save"
  9. File name: OraChope-Business-Card-PRINT.pdf

🔍 Browser Console Check

How to check browser console for errors:

  1. Windows/Linux: Press F12 or Ctrl+Shift+I
  2. Mac: Press Cmd+Option+I
  3. Click the "Console" tab
  4. Look for red error messages

❌ If you see these errors, iframe is BLOCKED:

  • Refused to display in a frame because it set 'X-Frame-Options' to 'deny'
  • Refused to display in a frame because it set 'X-Frame-Options' to 'sameorigin'
  • Refused to frame because it violates the following Content Security Policy

✅ If no errors and you see content, iframe WORKS!
